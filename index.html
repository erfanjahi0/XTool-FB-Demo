📋 Quick Setup

  1. Open Chrome → chrome://extensions → enable Developer Mode
  2. Click Load unpacked → select the extension/ folder
  3. Copy the Extension ID → paste it into js/extension-bridge.js as EXTENSION_ID
  4. Log in to facebook.com in the same Chrome profile
  5. Serve locally: python3 -m http.server 8000 then open localhost:8000
  6. OR deploy to Vercel — see DEPLOY.md in the project root

For personal use on your own Facebook account and pages only.
